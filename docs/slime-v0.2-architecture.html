Slime v0.2 Architecture

Self-Evolving Electron App — 内置 LLM 网关 · 多渠道聚合 · 协议转换 · 统计分析

Gateway Core (Route + Relay)

  • • Router: Group 名称 → 候选 Channel 列表
  • • Balancer: 轮询 / 随机 / 故障转移 / 加权
  • • CircuitBreaker: 三态熔断 + 指数退避
  • • KeyPool: 多 Key 轮转，选消费最少的
  • • ModelRegistry: Slot → Group 映射
  • • StatsCollector: 内存 Buffer → 30s 批量刷盘
  • • Outbound Adapters: 内部格式 → Provider 协议
  • • Relay Loop: route → key → outbound → retry

数据层 (SQLite)

  • • better-sqlite3 同步 API
  • • 分级存储: 7d 逐条 / 30d 小时聚合 / 90d 天聚合
  • • 稳态 2-6 MB（日均 ~1000 次请求）
  • • 费用 = Σ(tokens × price) / 1M
  • • 预设价格 + 手动覆盖

双入口接入

  • • 内部: AgentPresenter → Anthropic → localhost
  • • 外部: Claude Code / Codex → HTTP API
  • • 入站: Anthropic Messages + OAI Responses
  • • 出站: Anthropic / OpenAI / Gemini / DeepSeek / ...
  • • Slot: auto / lite / pro / max / chat / image

UI 布局

  • • 侧边栏新增"网关"入口
  • • 单页 Tab: 渠道 / 分组 / 接入 / 日志
  • • 顶部指标卡片常驻（请求/Token/费用/缓存）
  • • 全局设置: 端口 / 熔断参数 / 数据保留 / Slot 映射
  • • Onboarding 4 步: Welcome → Channel → Slot → Identity
